Introduction to Marks

Set mark 'a' here using 'ma' command. Marks allow you to save specific positions in your document and return to them quickly using single-quote or backtick navigation.

Understanding Vim Marks

Marks are bookmarks that save cursor positions. Use lowercase letters (a-z) for marks within a file, and uppercase letters (A-Z) for marks that persist across files and Vim sessions.

  • ma - Set mark 'a' at current position
  • 'a - Jump to line of mark 'a'
  • `a - Jump to exact position of mark 'a'
  • :marks - List all marks
  • :delmarks a - Delete mark 'a'

Session Persistence (mark 'E')

Global marks persist across Vim sessions when viminfo is properly configured. This enables consistent navigation patterns across development sessions.

Configuration (mark 'F')

" Viminfo configuration for mark persistence
set viminfo='100,f1,<500,:100,@100,/100,h

" Where:
" '100 = save marks for last 100 files
" f1   = store global marks A-Z
" <500 = save 500 lines for each register
" :100 = save 100 lines of command history
" @100 = save 100 lines of input history
" /100 = save 100 search patterns
" h    = disable hlsearch when loading viminfo
