Understanding Vim Modes

Vim has several modes that serve different purposes:

Normal Mode

The default mode for navigation and commands

Insert Mode

For typing and editing text content

Visual Mode

For selecting and manipulating text blocks

Command Mode

For executing vim commands and operations

Quick Reference

  • i - Insert mode before cursor
  • a - Insert mode after cursor
  • o - New line below and insert mode
  • v - Visual mode
  • : - Command mode
  • Esc - Return to Normal mode

Pro Tip

Always remember which mode you're in. The status line usually shows the current mode.
