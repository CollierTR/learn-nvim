Character Deletion

Delete single characters with precision using x and X commands. Use x to delete the character under the cursor.

  • x - Delete character under cursor
  • X - Delete character before cursor
  • nx - Delete n characters forward
  • x - Delete character under cursor
  • X - Delete character before cursor
  • Precise character-level control
  • Combine with counts for multiple deletions

Word Deletion

Delete entire words efficiently with dw, db, and related commands.

  • dw - Delete word forward
  • db - Delete word backward
  • de - Delete to end of word
  • daw - Delete around word (including spaces)
  • diw - Delete inner word (excluding spaces)
  • These commands work on word boundaries
  • dw - Delete word forward
  • db - Delete word backward
  • de - Delete to end of word

Line Deletion

Advanced Line Operations

Delete entire lines or parts of lines with powerful line-based commands. These are some of the most powerful commands in Vim.

  • dd - Delete entire line
  • D or d$ - Delete to end of line
  • d0 - Delete to beginning of line
  • dG - Delete to end of file
  • dgg - Delete to beginning of file
  • Tip: Use counts like 5dd to delete 5 lines
