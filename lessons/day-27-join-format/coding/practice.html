Formatting Settings and Commands

Essential Commands

Join Commands

  • J - Join current line with next
  • 3J - Join 3 lines starting from current
  • gJ - Join without adding spaces
  • :join - Join lines in range

Format Commands

  • gq - Format selected text
  • gqap - Format current paragraph
  • gqG - Format to end of file
  • gw - Format and keep cursor position

Settings

  • :set textwidth=80 - Set line width
  • :set formatoptions - Format options
  • :set wrap - Enable visual wrapping
  • :set linebreak - Break at words
