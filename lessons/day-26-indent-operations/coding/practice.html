Indentation Commands and Tips

Basic Commands

  • >> - Indent current line
  • << - Unindent current line
  • > - Indent selection
  • < - Unindent selection
  • == - Auto-indent current line

Visual Mode

  • Select lines with V
  • Use > or < to indent/unindent
  • Repeat with . command
  • Stay in visual mode after indent
  • Use gv to reselect last selection

Advanced Techniques

  • =G - Auto-indent to end of file
  • =% - Auto-indent matched block
  • 3>> - Indent 3 lines
  • >i{ - Indent inside braces
  • >ap - Indent around paragraph

Master indentation commands for clean, readable code formatting.

Commands: >>/<< (line), >/< (selection), == (auto-indent), =G (indent to end)
