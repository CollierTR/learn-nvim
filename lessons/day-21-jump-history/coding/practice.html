Understanding Jump History

Vim maintains a jump list that tracks significant cursor movements. Jumps include:

  • Moving to a different line with G, gg, or line numbers
  • Search operations with / or ?
  • Using % to match brackets
  • Jumping to marks with ' or `
  • Using tags or other navigation commands

About Jump Navigation

Jump navigation is one of Vim's most powerful features for maintaining context while exploring code or documents. When you make a significant movement, Vim automatically adds your previous position to the jump list.

How Jump Lists Work

The jump list is a stack-like structure, but it's more sophisticated than a simple stack. Vim maintains separate jump lists for each window, and the lists can hold up to 100 entries by default.

When you make a jump, your current position is added to the list before moving. This means you can always return to where you came from, even after multiple jumps.

Essential Jump Commands

  • Ctrl+o - Go to previous jump location
  • Ctrl+i - Go to next jump location (forward in history)
  • :jumps - Display the jump list
  • '' - Jump to last jump position
  • '. - Jump to last change position
