function calculateTotal(items) {
    let total = 0;
    for (let i = 0; i < items.length; i++) {
        total += items[i].price;
    }
    return total;
}


function processOrder(order) {
    const items = order.items;
    const total = calculateTotal(items);
    const tax = total * 0.1;
    const finalTotal = total + tax;

    return {
        subtotal: total,
        tax: tax,
        total: finalTotal
    };
}


class ShoppingCart {
    constructor() {
        this.items = [];
        this.total = 0;
    }

    addItem(item) {
        this.items.push(item);
        this.updateTotal();
    }

    removeItem(itemId) {
        this.items = this.items.filter(item => item.id !== itemId);
        this.updateTotal();
    }

    updateTotal() {
        this.total = calculateTotal(this.items);
    }
}

Article Content for Line Practice

Understanding Vim Line Operations (Line 410)

Line 412: Vim's line-based operations are fundamental to efficient text editing. Understanding how to navigate, manipulate, and work with lines will significantly improve your productivity.

Basic Line Navigation (Line 415)

Line 417: The most basic line navigation commands are j (down) and k (up). These single-character commands allow you to move through your document line by line.

Line 419: For faster navigation, you can prefix these commands with numbers. For example, 10j moves down 10 lines, while 5k moves up 5 lines.

Direct Line Jumping (Line 422)

Line 424: When you know exactly which line you want to reach, direct jumping is more efficient than relative movement.

Line 426: The :line_number command (like :100) takes you directly to that line. Alternatively, you can use line_numberG (like 100G) for the same result.

Line 428: Special cases include gg (first line) and G (last line) for document boundaries.

Line Manipulation Commands (Line 431)

Line 433: Once you can navigate efficiently, line manipulation becomes powerful. The dd command deletes an entire line, while yy copies it.

Line 435: Pasting works with p (paste below) and P (paste above). These commands work in conjunction with copying and cutting operations.

Line 437: Creating new lines is handled by o (open below) and O (open above), both of which enter insert mode on the new line.

Advanced Techniques (Line 440)

Line 442: Combining line operations with repetition creates powerful workflows. For instance, 3dd deletes 3 lines, while 5yy copies 5 lines.

Line 444: Visual line mode (V) allows you to select multiple lines for operations, extending these basic commands to work on blocks of text.

Line 446: Practice these commands regularly to build muscle memory and increase your editing efficiency.
