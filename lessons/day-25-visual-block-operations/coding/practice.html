Visual Block Tips and Techniques

Common Patterns

  • Use Ctrl+v to enter visual block mode
  • Select columns with h/j/k/l movements
  • Use I to insert at beginning of selection
  • Use A to append at end of selection
  • Use c to change selected block
  • Use r to replace characters in block
  • Use > and < to indent/unindent blocks
