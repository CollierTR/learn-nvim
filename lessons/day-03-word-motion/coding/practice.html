Understanding Different Word Types

lowercase words (w, b, e)

Navigate by alphanumeric words separated by non-alphanumeric characters 

const user_profile_data = {
  first_name: 'John',
  last_name: 'Doe',
  email_address: 'john.doe@example.com'
};

Each highlighted section represents a single 'word' for w/b/e commands

UPPERCASE WORDS (W, B, E)

Navigate by whitespace-separated sequences, ignoring punctuation within 

const user_profile_data = {
  first_name: 'John',
  last_name: 'Doe',
  email_address: 'john.doe@example.com'
};

Each highlighted block represents a single 'WORD' for W/B/E commands

Word Motion Quick Reference

Command Direction Word Boundary Cursor Position Example Usage
w Forward Alphanumeric Beginning of word user_name ‚Üí _name
b Backward Alphanumeric Beginning of word user_name ‚Üê user_
e Forward Alphanumeric End of word user_name ‚Üí user_
ge Backward Alphanumeric End of word user_name ‚Üê name
W Forward Whitespace Beginning of WORD user_name = 'value';
B Backward Whitespace Beginning of WORD = 'value'; ‚Üê user_name
E Forward Whitespace End of WORD user_name ‚Üí name
gE Backward Whitespace End of WORD 'value'; ‚Üê 'value';

Practice Tips

Basic Word Motion

  • Start with w and b for basic navigation
  • Use e when you need to reach word endings
  • Practice ge for backward end movement

Advanced WORD Motion

  • Use W/B/E for punctuation-heavy code
  • WORD commands ignore underscores and hyphens
  • gE is rarely used but can be helpful
