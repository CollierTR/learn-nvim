Bracket Matching Practice

Mathematical expressions: (a + b) * (c - d) / (e + f)

Array access: items[0] + data[index] - values[key]

Object notation: {name: "value", count: 42, active: true}

Function calls: calculate(x, y) + process(a, b, c)

Complex nesting: {users: [{id: 1, data: [a, b, c]}, {id: 2, items: (x + y)}]}

Multiple levels: ((a + b) * (c + d)) / ((e - f) * (g + h))

Mixed brackets: [item(x), func{a: b}, calc(y + z)]
